The maiTea Bot allows you to view your maimai FiNALE (and older) profile, scores, and other data on Discord, alongside various other tools. You can see your own data, or another persons, by submitting their username.

When you sign up to the bot, anyone else with access to the bot can lookup your data assuming they have your Discord profile.

Please note that, while this bot is in the official maiTea Discord server, it is not ran, moderated, or developed by maiTea themselves. It is a fan-made project.

Security and Data Handling

maiTea API tokens offer very limited functionality, specifically read-only functions for reading scores, profiles, plays, and some other information. Profiles can not be written to, edited, or deleted using these tokens. For further information on the API, please read the maiTea API documentation.

Due to the nature of the information stored, and it not being sensitive, it is stored in plain-text JSON alongside the project's code. The only information stored is the following:
API access token, Discord user ID, maimai profile name, the latest time the profile got updated, the rating, and peak rating of the profile.

It is not shared with anyone else, and cannot be accessed by anyone else. Access codes and other information not included above is not stored. If you are unhappy with this, and would like your information removed, please contact me in the maiTea Discord.

Bot Permissions

When adding the bot to your user account, the only permissions accessed are app commands access. The bot is unable to read information about your profile other than basic information, and it is not possible to identify who exactly has added it.

The default invite link contains the permission code 139586817088, which only allows read/send messages, and external emoji uses, alongside some others. There are no moderation/administrator permissions used. This permission code is not required for the bot to be used, unless owner-specific non-slash commands are required in the server.
